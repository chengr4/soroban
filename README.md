## IRR (內部報酬率)

IRR 是利用總存入與總支出來計算年化報酬率

eg:

- at first 3 years, each year save 100k, the next 3 years, each - year withdraw 110k
needs discount (貼現)
- 100k + 100K/(1+IRR) + 100K/(1+IRR)^2 - 110K/(1+IRR)^3 - 110K/(1+IRR)^4 - 110K/(1+IRR)^5 = 0
- IRR = 3.23%
